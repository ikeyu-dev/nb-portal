# NB-Portal 機能まとめ

## ホーム（ダッシュボード）

- 今日の日付・時計・天気を表示
- 今後のスケジュール一覧
- 本日の欠席者一覧

## カレンダー

- 月表示カレンダーで予定を一覧表示
- 予定の登録・編集・削除
- 複数日にまたがる予定の登録（終日イベント）

## 欠席連絡

- 活動への欠席・遅刻・早退を連絡
- 中抜け時間の指定も可能

## 機材管理

- マイク・スピーカー・ケーブルなどの機材一覧
- カテゴリ別フィルタリング（MIC/SPK/CAB/OTH）
- 機材の新規登録（同一機材の複数登録に対応）
- 機材名の編集
- 機材の削除

## バスダイヤ

- 東武公・新白岡のバス時刻表を表示
- 日付を指定して確認可能

## 部会メモ

- 統一フォーマットで部会メモを作成
- マークダウン形式でコピー
- Discordなどに貼り付け可能

## お知らせ

- カレンダーの追加・更新・削除履歴を表示
- 誰がいつ何を更新したかを確認
- プッシュ通知で予定の変更をリアルタイム通知

## ヘルプ

- 各機能の使い方を確認
- デモ付きで操作方法を学べる
