# NB-Portal 機能まとめ

## ホーム（ダッシュボード）

- 今日の日付・時計・天気を表示
- 今後のスケジュール一覧
- 本日の欠席者一覧

## カレンダー

- 月表示カレンダーで予定を一覧表示
- 予定の登録・編集・削除
- 複数日にまたがる予定の登録（終日イベント）

## 欠席連絡

- 活動への欠席・遅刻・早退を連絡
- 中抜け時間の指定も可能

## 備品管理

- マイク・スピーカー・ケーブルなどの備品一覧
- カテゴリ別フィルタリング

## バスダイヤ

- 東武公・新白岡のバス時刻表を表示
- 日付を指定して確認可能

## お知らせ

- カレンダーの追加・更新履歴を表示
- 誰がいつ何を更新したかを確認
