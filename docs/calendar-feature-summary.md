# NB-Portal 機能まとめ
<span style="display:block;height:3px;background:#2a83a2;margin-top:-8px;margin-bottom:16px;"></span>

## ホーム（ダッシュボード）
<span style="display:block;height:2px;background:#2a83a2;margin-top:-8px;margin-bottom:12px;"></span>

- 日付・時計・天気を表示
- 今後のスケジュール一覧
- 本日の欠席者一覧

## カレンダー
<span style="display:block;height:2px;background:#2a83a2;margin-top:-8px;margin-bottom:12px;"></span>

- 月表示で予定一覧
- 予定の登録・編集・削除
- 複数日にまたがる予定（終日イベント）

## 欠席連絡
<span style="display:block;height:2px;background:#2a83a2;margin-top:-8px;margin-bottom:12px;"></span>

- 欠席・遅刻・早退・中抜けを連絡
- 中抜け時間の指定も可能

## 機材管理
<span style="display:block;height:2px;background:#2a83a2;margin-top:-8px;margin-bottom:12px;"></span>

- マイク・スピーカー・ケーブルなどの機材一覧
- カテゴリ別フィルタリング（MIC/SPK/CAB/OTH）
- 新規登録（同一機材の複数登録対応）
- 編集・削除

## バスダイヤ
<span style="display:block;height:2px;background:#2a83a2;margin-top:-8px;margin-bottom:12px;"></span>

- 東武公・新白岡のバス時刻表を表示
- 日付指定で確認可能

## 部会メモ
<span style="display:block;height:2px;background:#2a83a2;margin-top:-8px;margin-bottom:12px;"></span>

- 統一フォーマットで部会メモを作成
- マークダウン形式でコピー
- Discordなどに貼り付け可能

## お知らせ
<span style="display:block;height:2px;background:#2a83a2;margin-top:-8px;margin-bottom:12px;"></span>

- カレンダーの追加・更新・削除履歴を表示
- 誰がいつ何を更新したかを確認
- プッシュ通知でリアルタイム通知
